## 📊 安装方法

### 方法一：克隆整个技能库

```bash
# 克隆仓库
git clone https://github.com/huohongxu/openclaw-skills.git ~/.qclaw/workspace/skills

# 进入目录
cd ~/.qclaw/workspace/skills
```

### 方法二：使用 SkillHub 安装单个技能

```bash
# 查看可用技能
skillhub search <关键词>

# 安装单个技能
skillhub install <skill-name>
```
